 Car Parts Dispatch System (C)

## Description
C program to process and manage car parts orders. Features sorting by weight, merging team lists, efficient product search, and delivery summary generation using optimized algorithms for fast warehouse order handling.

## Features
- Sort items by weight using O(N log N) algorithm
- Merge 4 team order files into a single dispatch list (O(N))
- Search for earliest occurrence of a product by weight (O(log N))
- Generate delivery summary for all vans (O(N))
